NOTIFICATION SOUND FILES
========================

Folder ini untuk menyimpan file audio notifikasi.

CARA TAMBAH SOUND:
1. Download sound notification gratis dari:
   - https://notificationsounds.com/
   - https://freesound.org/
   - https://mixkit.co/free-sound-effects/notification/

2. Atau buat sendiri di:
   - https://sfxr.me/ (8-bit sounds)
   - Audacity (free audio editor)

3. Format yang didukung:
   - notification.mp3 (recommended)
   - notification.ogg (fallback)
   - notification.wav (fallback)

4. Copy file ke folder ini dengan nama:
   notification.mp3
   notification.ogg

RECOMMENDED SOUNDS:
- "Ding" sound untuk pesanan baru
- "Bell" sound untuk order ready
- "Alert" sound untuk urgent orders

Jika tidak ada file sound, sistem akan tetap bekerja tanpa audio.
Browser notification tetap jalan!

TESTING:
Buka kitchen display dan test notifikasi:
- Browser akan minta permission untuk notification
- Klik "Allow"
- Buat order baru dari kasir/customer
- Dapur akan dapat notif dalam 5 detik

Note: Sound hanya play jika user sudah interact dengan page
(Chrome/Safari security policy)
