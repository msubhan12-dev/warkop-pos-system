=====================================================
🐛 TROUBLESHOOTING:
=====================================================

❌ "Username atau password salah"
→ Pastikan username: admin (lowercase)
→ Pastikan password: password (lowercase)
→ Clear browser cache

❌ Database connection error
→ Pastikan MySQL running di XAMPP
→ Database warkop_db sudah diimport

❌ Blank page
→ Cek PHP error di XAMPP log
→ Pastikan semua file ada
